
# 電商營運分析平台 Dashboard Spec v1

## 一、產品定位

**產品名稱**
電商營運分析平台

**用途**
提供公司內部營運、CRM、行銷與商品團隊使用的電商分析 Dashboard。

**主要目的**
1. 即時監控營運狀況  
2. 分析會員價值與回購行為  
3. 評估商品與流量成效  
4. 協助 CRM 制定會員經營策略  

**主要使用者**
- 營運團隊
- CRM 團隊
- 行銷投放團隊
- 商品管理團隊
- 管理層

# 三、全站共用控制列

支援選項

- 最近7天
- 最近30天
- 最近90天
- 今年
